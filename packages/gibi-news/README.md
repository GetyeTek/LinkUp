# @linkup/gibi-news
**Maintainer:** LinkUp Discovery & Content Squad

This micro-frontend handles global news, event discovery, and organizational updates. It operates completely independently of the core academic engine, ensuring content delivery remains fast and unblocked.

### Architecture
- Renders full-screen immersive snap feeds.
- Intercepts local scrolling arrays.
- Consumes `@linkup/core-sdk` for auth token injection.
